# Linear Regression From Scratch

This project implements **Linear Regression** from scratch using Python and NumPy. The model learns the relationship between input and output data using **gradient descent** and can predict unseen values.

## Features
- Linear Regression implemented manually
- Gradient descent for parameter optimization
- Data visualization using matplotlib
- Predict new values
